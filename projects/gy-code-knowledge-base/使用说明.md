## 界面说明

### 📚 知识库页（默认）

| 左侧面板 | 功能 |
|---------|------|
| 🤖 **AI 代码搜索** | 中文自然语言搜代码 |
| 📂 **语言/难度筛选** | 按语言或难度筛选代码 |
| 📚 **知识库问答** | 上传文档 → 向量检索 → AI 回答 |

| 右侧操作 | 说明 |
|---------|------|
| 查看代码 | 点击卡片展开，语法高亮 |
| 新增/编辑/删除 | 点击对应按钮操作 |
| 复制代码 | 展开后点 📋 复制 |

### 💬 AI 对话页

点击顶部 **「💬 AI 对话」** 切换到 AI 代码解析助手。

| 功能 | 说明 |
|-----|------|
| 🆕 **创建对话** | 点击「➕ 新对话」开始新会话 |
| ✏️ **重命名** | 点击对话标题旁的 ✏️ 按钮 |
| 🗑️ **删除对话** | 鼠标悬停会话，点右上 ✕ 删除 |
| 🧠 **查看提示词** | 每条 AI 回复右上角的 🧠 按钮，可查看完整 system prompt 和调用参数 |
| 💬 **多轮对话** | 同一对话内可连续追问，AI 保留上下文 |
| 📜 **历史回溯** | 切换对话自动加载历史消息 |

**代码解析能力：**
- 逐行解释代码逻辑和意图
- 分析时间/空间复杂度
- 识别算法、数据结构和设计模式
- 调试帮助和优化建议
- 编程概念学习和指导

## API 接口

### 代码片段管理
```
GET    /api/code_snippets             查询代码（支持筛选/排序/分页）
GET    /api/code_snippets/:id         获取单条
POST   /api/code_snippets             新增
PUT    /api/code_snippets/:id         修改
DELETE /api/code_snippets/:id         删除
GET    /api/languages                 编程语言列表
```

### AI 搜索 & 知识库
```
POST   /api/ai/search                 AI 自然语言搜代码
POST   /api/rag/documents             上传文档到知识库
GET    /api/rag/documents             查看知识库文档
DELETE /api/rag/documents/:title      删除文档
GET    /api/rag/export                导出知识库（Markdown）
POST   /api/rag/ask                   基于知识库问答
```

### 💬 AI 代码解析对话（新增）
```
POST   /api/chat/sessions             创建新对话
GET    /api/chat/sessions             对话列表（含最后消息预览）
PATCH  /api/chat/sessions/:id         重命名对话
DELETE /api/chat/sessions/:id         删除对话（级联删除消息）
GET    /api/chat/sessions/:id/messages 查看对话历史消息
POST   /api/chat/sessions/:id/chat    发送消息 → AI 回复（含提示词记录）
```

**调用示例：**
```bash
# 查所有代码
curl http://localhost:3000/api/code_snippets

# AI 搜索
curl -X POST http://localhost:3000/api/ai/search \
  -H "Content-Type: application/json" \
  -d '{"query":"找排序算法的代码"}'

# 知识库提问
curl -X POST http://localhost:3000/api/rag/ask \
  -H "Content-Type: application/json" \
  -d '{"query":"这个项目怎么用？"}'

# 上传文档
curl -X POST http://localhost:3000/api/rag/documents \
  -H "Content-Type: application/json" \
  -d '{"title":"文档标题","content":"文档内容...","source":"来源"}'

# 导出知识库
curl -o knowledge-base.md http://localhost:3000/api/rag/export

# (新) 创建 AI 对话
curl -X POST http://localhost:3000/api/chat/sessions \
  -H "Content-Type: application/json" \
  -d '{"title":"代码解析"}'

# (新) 发送消息给 AI
curl -X POST http://localhost:3000/api/chat/sessions/1/chat \
  -H "Content-Type: application/json" \
  -d '{"message":"帮我解析这段 Python 冒泡排序"}'

# (新) 查看对话历史
curl http://localhost:3000/api/chat/sessions/1/messages
```

**查询参数：** `?language=Python` 按语言筛选 · `?_sort=created_at&_order=DESC` 排序 · `?_page=1&_size=20` 分页

## 常见问题

| 问题 | 解决 |
|-----|------|
| 端口被占用 | `taskkill /f /im node.exe` 再 `npm start` |
| AI 搜索报错 | 检查 `.env` 中 DEEPSEEK_API_KEY 是否正确 |
| 知识库为空 | 点击虚线框上传 `.txt` / `.md` 文件 |
| AI 对话无响应 | 检查终端日志是否有 API 报错，确认 `.env` 文件在项目根目录 |
| 对话历史丢失 | 不会丢失，数据存 MySQL，刷新页面后重新进入即可 |
| 页面空白 | 确认终端没关，刷新页面 |
| 中文乱码 | 终端显示问题，网页正常 |
| 重置数据库 | `mysql -u api_user -papi_pass_2024 -e "DROP DATABASE user_api"` 再重启 |
