## API 接口

```
GET    /api/code_snippets             查询代码（支持筛选/排序/分页）
GET    /api/code_snippets/:id         获取单条
POST   /api/code_snippets             新增
PUT    /api/code_snippets/:id         修改
DELETE /api/code_snippets/:id         删除
GET    /api/languages                 编程语言列表
POST   /api/ai/search                 AI 自然语言搜代码
POST   /api/rag/documents             上传文档到知识库
GET    /api/rag/documents             查看知识库文档
DELETE /api/rag/documents/:title      删除文档
GET    /api/rag/export                导出知识库（Markdown）
POST   /api/rag/ask                   基于知识库问答
```

**调用示例：**
```bash
# 查所有代码
curl http://localhost:3000/api/code_snippets

# AI 搜索
curl -X POST http://localhost:3000/api/ai/search \
  -H "Content-Type: application/json" \
  -d '{"query":"找排序算法的代码"}'

# 知识库提问
curl -X POST http://localhost:3000/api/rag/ask \
  -H "Content-Type: application/json" \
  -d '{"query":"这个项目怎么用？"}'

# 上传文档
curl -X POST http://localhost:3000/api/rag/documents \
  -H "Content-Type: application/json" \
  -d '{"title":"文档标题","content":"文档内容...","source":"来源"}'

# 导出知识库
curl -o knowledge-base.md http://localhost:3000/api/rag/export
```

**查询参数：** `?language=Python` 按语言筛选 · `?_sort=created_at&_order=DESC` 排序 · `?_page=1&_size=20` 分页

## 常见问题

| 问题 | 解决 |
|-----|------|
| 端口被占用 | `taskkill /f /im node.exe` 再 `npm start` |
| AI 搜索报错 | 检查 `.env` 中 DEEPSEEK_API_KEY 是否正确 |
| 知识库为空 | 点击虚线框上传 `.txt` / `.md` 文件 |
| 页面空白 | 确认终端没关，刷新页面 |
| 中文乱码 | 终端显示问题，网页正常 |
| 重置数据库 | `mysql -u api_user -papi_pass_2024 -e "DROP DATABASE user_api"` 再重启 |
